# 高级签到插件

一个功能丰富的QQ签到插件，基于AstrBot框架开发。

## 功能特性

1. **基础签到系统**
   - 每日签到获得金币和经验
   - 连续签到奖励机制
   - 等级系统

2. **等级系统**
   - 初始等级1级，升级所需经验递增
   - 经验奖励随等级提升
   - 金币奖励随等级提升

3. **排行榜系统**
   - 总签到排行榜
   - 连续签到排行榜
   - 等级排行榜
   - 世界总签到排行榜

4. **商店系统**
   - 购买补签卡
   - 使用补签卡补签
   - 背包管理

## 使用方法

### 签到命令

- `/签到` - 每日签到
- `/个人信息` - 查看个人信息
- `/总排行榜` - 总签到排行榜
- `/连续签到排行榜` - 连续签到排行榜
- `/等级排行榜` - 等级排行榜
- `/世界排行榜` - 世界总签到排行榜
- `/购买 补签卡 数量` - 购买补签卡
- `/补签 1` - 使用补签卡补签
- `/查看背包` - 查看背包物品

### 奖励机制

#### 经验奖励
- 基础经验：10-50点
- 等级加成：每级增加15%
- 连续签到奖励：
  - 连续7天：升级所需经验的3%
  - 连续30天：升级所需经验的13%
  - 连续30天额外奖励：升级所需经验的3%

#### 金币奖励
- 基础金币：90-180枚
- 等级加成：每级增加15%
- 连续签到奖励：
  - 连续7天：金币的3%
  - 连续30天：金币的13%
  - 连续30天额外奖励：金币的3%

### 等级系统

- 初始等级：1级
- 1级升2级：200经验
- 每级所需经验：上一级的120%

### 商店系统

- 补签卡：1000金币/张
- 补签卡使用规则：
  - 只能补签前三天内未签到的日子
  - 最多储存10张补签卡
  - 补签按顺序进行：昨天、前天、大前天
